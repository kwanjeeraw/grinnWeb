Grinn

Grinn is a bioinformatics platform contains a graph database, an R package and web application for metabolomics studies. It aims to broaden metabolomics studies by allowing mapping of omics data such as genomics, transcriptomics, proteomics and metabolomics for a systems level exploration.

Features

There are three main functions:

  • SEARCH - query for metabolites, proteins, genes and metabolic pathways
  • BUILD - connect given metabolites based on several kinds of relationships
  • PAIR - generate an integrated network of metabolites, proteins, genes and pathways
