SEARCH

SEARCH is to query for elements such as metabolites, proteins, genes and metabolic pathways.

1) SEARCH for 6 metabolites by name, each keyword is separated with a new line.
NOTE: Grinn supports search by other database ID by providing the keyword in following format: databaseName:databaseID.


2) SEARCH result is a table of matched keywords with the list of incoming and outgoing relationships of the queried keywords. Click a name will direct to the information page.
NOTE: Information of incoming and outgoing relationships includes element name, relationship type and relationship property inside curly brackets (if exist).
